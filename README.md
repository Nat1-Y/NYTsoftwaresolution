## 🚀 Getting Started Locally

You can preview the portfolio page instantly. Since it relies on vanilla HTML/CSS/JS, it runs with zero installation overhead:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Nat1-Y/NYTsoftwaresolutions.git
   cd NYTsoftwaresolutions
   ```
2. **Launch a server**:
   * Using Python:
     ```bash
     python -m http.server 8000
     ```
   * Or simply double-click `index.html` to open it in your browser.
